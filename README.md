 Classification & Regression Project

A simple Python project implementing Supervised Machine Learning models.

 📁 Repository Structure
* datasets: Folder containing the data files.
* classification_code.py: Python code for classification models.
* regression_code.py: Python code for regression models.
* main.py: The main script to run the project.

 🛠️ Tech Stack
* Python
* Machine Learning (Supervised Learning)
